# Roozerball

Python Roozerball engine and a minimal desktop GUI.

## Run the GUI

```bash
python -m roozerball.gui
```

## Current slice

- Phase-by-phase game loop for clock, ball, initiative, movement, combat, and scoring
- Auto-generated teams and starting positions
- Tkinter board viewer with scoreboard, initiative display, movement highlighting, selected-figure HUD, and replay log
